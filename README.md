# ♟️ Grandmaster Chess

 ![React](https://img.shields.io/badge/-React-blue?style=flat-square) ![Node.js](https://img.shields.io/badge/-Node.js-blue?style=flat-square) ![Productivity](https://img.shields.io/badge/-Productivity-blue?style=flat-square) ![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

A premium, web-based strategy chess game featuring an advanced piece-validation system and glassmorphism interface design.

## 📝 Overview
Grandmaster Chess brings the FIDE chess rules to your browser without the need for external server infrastructure. The engine perfectly validates moves like knight-L shapes and infinite-range pathfinding in real-time. Additionally, enjoy an automated move-logger that processes exact match states.

## ✨ Key Features
- **Strict FIDE Compliance**: Legal move generation, pawn specific double moves, capturing physics.
- **Vs AI Module**: Supports Player vs Computer mode built with an integrated heuristic evaluation model.
- **Stunning UI/UX**: Premium "Glassmorphism" UI giving it a futuristic touch compared to classic wood.
- **Game Analytics**: Realtime move history and tracked score/timer badges for competitive feels.
- **Responsive Web Design**: Dynamically shapes the board to fit 4k monitors or compact smartphones alike.

## 🛠 Tech Stack
- Frontend: `HTML5`, `Vanilla CSS3`
- Game Logic Engine: `ES6+ JavaScript`
- Architecture: Functional 2D Matrix Grid

## 🚀 Getting Started
Since this is a client-side vanilla application, setting it up is incredibly agile.

1. Clone the repository to your local machine.
2. Open `index.html` directly in any modern Web Browser (Chrome, Firefox, Safari).
3. Play!


## 🌐 Deployment

### Vercel / Netlify
1. This is a standalone static site.
2. Drag and drop the project folder into Netlify Drop or connect via GitHub.
3. No build command is required. Just ensure `index.html` is in the root.
